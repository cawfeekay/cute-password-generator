# Cute Password Generator

A smol & cute password generator I made for fun back in 2022.
I wanted to make something less boring than the usual plain password tools with a pink pastel aesthetic.

### Features
- adjustable password length (8-24)
- toggle uppercase, numbers, and symbols
- one click copy
- looks cute

live demo: https://cawfeekay.github.io/cute-password-generator/
